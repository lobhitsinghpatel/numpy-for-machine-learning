# NumPy Arrays vs Python Lists

## Why NumPy?

NumPy is a numerical computing library used in Machine Learning.

## Python Lists

- Flexible
- Can store mixed data types
- More memory consumption
- Slower calculations

## NumPy Arrays

- Same data type
- Continuous memory allocation
- Faster computation
- Supports vectorization

## Vectorization

Vectorization performs operations on entire arrays without loops.

Example:

arr = np.array([1,2,3,4])

arr * 2

Output:

[2 4 6 8]

## Multi-Dimensional Arrays

1D Array → Vector

2D Array → Matrix

3D Array → Tensor
