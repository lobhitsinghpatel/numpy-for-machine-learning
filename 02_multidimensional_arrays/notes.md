# Multi-Dimensional Arrays in NumPy

## What are Multi-Dimensional Arrays?

Multi-dimensional arrays store data in rows and columns.

They are the foundation of Machine Learning datasets.

## Types of Arrays

### 1D Array (Vector)

Example:

[10, 20, 30]

Shape:

(3,)

Used for:
- Labels
- Single Features

---

### 2D Array (Matrix)

Example:

[
 [25,30000,2],
 [30,50000,5]
]

Shape:

(2,3)

Rows = Samples

Columns = Features

Used for:
- ML Datasets
- Data Analysis

---

### 3D Array (Tensor)

Example:

[
 [[1,2],[3,4]],
 [[5,6],[7,8]]
]

Shape:

(2,2,2)

Used for:
- Images
- Videos
- Deep Learning
